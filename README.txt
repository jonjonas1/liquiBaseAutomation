Running automated TCs using Eclipse
2 way to run it from class or from .xml file

* From Eclips/InteliJ
      1st Way
* expand the package "src/test/java"
* expand "/com.liquiBase" package
* right click on the desired TC class run as > TestNG
* browser should open and run the automation
* should show on Console page as well. 
      2nd way
^ from Eclipse/InteliJ
* right click on runByClassName.xml and run by > TestNg


Adding new TCs to the test suites
* expand the package "src/test/java"
* right click on "com.liquiBase" package
* Select New > Class
* Name your class accordingly and Finish
